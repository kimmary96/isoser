# Slack용 배포 체크리스트 / 승인 템플릿

아래 문구는 운영 채널에 그대로 붙여 넣어 사용할 수 있는 Slack 템플릿입니다.

## 1. 배포 시작 알림

```md
[배포 점검 시작]

- 환경:
- 버전 / 커밋:
- 점검자:
- 시작 시간:

오늘 배포 전 아래 핵심 항목을 확인합니다.
1. 로그인
2. 프로그램 탐색 / 비교
3. 프로필 / 활동 저장
4. AI 기능
5. PDF export

점검 완료 후 같은 스레드에 결과를 공유하겠습니다.
```

## 2. 점검 결과 공유 템플릿

```md
[배포 점검 결과]

- 환경:
- 버전 / 커밋:
- 점검자:
- 완료 시간:

- 첫 화면 / 공개 화면: 정상 / 이상
- 로그인 / 인증: 정상 / 이상
- 프로그램 탐색 / 비교: 정상 / 이상
- 프로필 / 활동 저장: 정상 / 이상
- AI 기능: 정상 / 이상
- PDF export: 정상 / 이상

특이사항:
- 
- 

최종 제안:
- 배포 가능 / 조건부 배포 / 배포 보류
```

## 3. 배포 승인 요청 템플릿

```md
[배포 승인 요청]

배포 전 점검을 완료했습니다.

- 환경:
- 버전 / 커밋:
- 점검 결과: 배포 가능
- 치명적 이슈: 없음

확인된 핵심 항목:
- 로그인 정상
- 프로그램 탐색 / 비교 정상
- 저장 정상
- AI 기능 정상
- PDF export 정상

이 스레드에서 승인 주시면 배포 진행하겠습니다.
```

```md
- 환경:
- 버전 / 커밋:

- 로그인: 정상
- 저장: 정상
- AI 기능: 정상
- PDF export: 정상

남은 이슈:
- 
- 

판단:
- 치명적 이슈는 아니므로 조건부 배포 가능

권장 후속 조치:
-

```

```md

- 환경:
- 버전 / 커밋:
- 점검자:

확인된 문제:
1. 
2. 

사용자 영향:
- 

현재 판단:
- 로그인 / 저장 / AI / 비교 / PDF 중 핵심 기능 이상

다음 조치:
- 원인 확인
- 수정 후 재검증
- 재승인 요청
```

## 6. 배포 완료 알림 템플릿

```md
[배포 완료]

- 환경:
- 버전 / 커밋:
- 완료 시간:

배포가 완료되었습니다.
배포 직전 점검 기준 핵심 기능은 정상 확인 상태입니다.

배포 후 모니터링 우선 항목:
1. 로그인
2. 업로드
3. AI 분석
4. PDF export
```
