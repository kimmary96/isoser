## 1. 첫 화면 / 공개 화면

- [ ] 메인 첫 화면이 정상적으로 열린다
- [ ] 상단 메뉴가 보인다
- [ ] `프로그램 탐색` 클릭이 된다
- [ ] `비교` 클릭이 된다
- [ ] `워크스페이스` 클릭이 된다
- [ ] `/programs` 페이지가 정상적으로 열린다
- [ ] `/compare` 페이지가 정상적으로 열린다

메모:

## 2. 로그인 / 인증

- [ ] 로그인 화면이 열린다
- [ ] Google 로그인 버튼이 동작한다
- [ ] 로그인 후 정상 화면으로 돌아온다
- [ ] 이미 로그인된 사용자는 다시 로그인 화면에 갇히지 않는다
- [ ] 비로그인 사용자가 보호 페이지 접근 시 로그인 화면으로 이동한다

메모:

## 3. 프로그램 탐색 / 비교

- [ ] 프로그램 목록이 비어 있지 않다
- [ ] 검색이 동작한다
- [ ] 필터가 동작한다
- [ ] 비교 페이지에서 3개까지 비교 가능하다
- [ ] 비교 관련도 계산이 정상 동작한다

메모:

## 4. 프로필 / 활동 저장

- [ ] 프로필 화면이 열린다
- [ ] 프로필 수정 후 저장된다
- [ ] 저장 후 새로고침해도 값이 유지된다
- [ ] 활동 목록이 열린다
- [ ] 새 활동 생성이 된다
- [ ] 활동 수정이 된다
- [ ] 활동 삭제가 된다
- [ ] 활동 이미지 업로드가 된다
- [ ] 비정상 파일 업로드 시 오류가 잘 보인다

메모:

## 5. AI 기능

- [ ] 공고 분석이 된다
- [ ] 요약 기능이 된다
- [ ] 코칭 기능이 된다
- [ ] 실패 시 무한 로딩이 아니라 오류 메시지가 보인다
- [ ] 짧은 시간 반복 요청 시 제한이 걸린다

메모:

## 6. 문서 / 출력

- [ ] 이력서 화면이 열린다
- [ ] 문서 저장이 된다
- [ ] PDF export가 된다

메모:

## 7. 운영 확인

- [ ] Google 로그인 실서비스 설정이 맞다
- [ ] 필수 환경변수 누락이 없다
- [ ] Upstash 사용 시 연결이 정상이다
- [ ] 에러 발생 시 로그가 남는다
- [ ] 민감정보가 로그에 직접 노출되지 않는다
