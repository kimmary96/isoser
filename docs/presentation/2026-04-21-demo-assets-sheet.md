# 2026-04-21 발표용 계정 / 샘플 데이터 준비 시트

## 문서 목적

- 발표 전에 필요한 계정, 샘플 PDF, 프로그램 URL, fallback 경로를 한 곳에 모은다.
- 실제 리허설 전에 비어 있는 항목이 없는지 빠르게 확인한다.
- 팀원들이 같은 자산을 기준으로 발표와 점검을 진행하게 한다.

## 사용 원칙

- 민감 정보는 문서에 직접 적지 않고, 위치 또는 관리 담당자만 적는다.
- 발표 전날까지 최소 1회, 발표 당일 시작 전에 1회 다시 확인한다.
- 값이 없으면 빈칸으로 두지 말고 `미정` 또는 `준비중`으로 적는다.
