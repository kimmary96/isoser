# 비개발자용 10분 런칭 체크리스트

이 문서는 개발 지식이 많지 않아도, 배포 직전 꼭 확인해야 하는 항목만 빠르게 점검할 수 있도록 정리한 문서입니다.

기술자용 상세 점검표는 [launch-smoke-test.md](./launch-smoke-test.md)를 참고합니다.

## 1. 먼저 확인할 결론

아래 4가지만 정상이라면, 현재 코드 기준으로는 런칭 준비가 된 상태에 가깝습니다.

1. 로그인된다
2. 프로그램 탐색이 된다
3. 문서/활동 저장이 된다
4. AI 분석이 된다

하나라도 막히면 바로 배포하지 말고 원인을 먼저 확인해야 합니다.

## 2. 10분 체크 순서

### 1단계. 첫 화면 확인

- 사이트 첫 화면이 정상적으로 열린다
- 상단 메뉴가 보인다
- `프로그램 탐색`, `비교`, `워크스페이스` 클릭이 된다

정상이면 다음 단계로 갑니다.

### 2단계. 로그인 확인

- `로그인` 또는 `무료로 시작하기`를 눌렀을 때 로그인 화면이 열린다
- Google 로그인 버튼이 정상 동작한다
- 로그인 후 다시 서비스 화면으로 돌아온다

문제 예시:
- 로그인 버튼을 눌러도 안 넘어감
- 로그인 후 빈 화면
- 로그인 후 계속 로그인 화면으로 돌아옴

이 중 하나라도 있으면 배포 보류입니다.

### 3단계. 프로그램 탐색 확인

- `/programs` 화면이 열린다
- 프로그램 목록이 비어 있지 않다
- 검색이나 필터를 바꿨을 때 화면이 정상 반응한다
- 비교 페이지(`/compare`)도 열린다

문제 예시:
- 프로그램이 하나도 안 나옴
- 비교 페이지가 깨짐
- 버튼이 눌리지 않음

### 4단계. 프로필 / 활동 저장 확인

- 로그인 후 `/dashboard/profile`이 열린다
- 이름이나 연락처를 수정하고 저장할 수 있다
- 활동 저장소(`/dashboard/activities`)가 열린다
- 새 활동을 만들고 저장할 수 있다
- 이미지 업로드가 된다

문제 예시:
- 저장 버튼을 눌러도 반응 없음
- 이미지 업로드 실패
- 저장 후 새로고침하면 내용이 사라짐

### 5단계. AI 기능 확인

- 공고 분석(`/dashboard/match`)이 동작한다
- 요약/코칭 기능이 동작한다
- 너무 오래 기다리지 않고 실패하면 실패 메시지가 나온다

문제 예시:
- 계속 로딩만 하고 끝나지 않음
- AI 응답이 비어 있음
- 요청할 때마다 에러만 남

### 6단계. 문서 출력 확인

- 이력서 화면(`/dashboard/resume`)이 열린다
- PDF export가 된다

문제 예시:
- PDF가 생성되지 않음
- 파일이 깨짐

## 3. 지금 배포하면 안 되는 신호

아래 중 하나라도 있으면 바로 배포하지 않는 것이 안전합니다.

- 로그인 실패
- 저장한 데이터가 새로고침 후 사라짐
- 이미지 업로드 실패
- AI 기능이 계속 실패
- 프로그램 목록이 비어 있음
- 비교 화면이 깨짐
- PDF export 실패

## 4. 지금 코드 기준으로 이미 보강된 것

현재 프로젝트에는 아래 안정화 작업이 반영되어 있습니다.

- 요청이 너무 많이 들어오면 자동 제한
- AI 응답이 너무 오래 걸리면 timeout 처리
- 업로드 파일이 정상 이미지인지 추가 확인
- 장애가 나면 운영 로그에 구조화된 기록 남김
- 이미지 렌더링 최적화 일부 적용

즉, 예전보다 “망가지기 쉬운 부분”은 많이 줄어든 상태입니다.

## 5. 아직 운영에서 꼭 확인할 것

코드가 괜찮아도 실제 서비스 환경에서는 아래를 꼭 봐야 합니다.

- Google 로그인 설정이 실제 도메인에서 맞는지
- 환경변수(`env`)가 빠진 것이 없는지
- Upstash를 쓰는 경우 연결이 실제로 되는지
- 실제 운영 데이터가 정상 조회되는지

## 6. 추천 최종 판정 방식

### 배포 가능

아래가 모두 만족되면 배포 가능으로 봐도 됩니다.

- 로그인 성공
- 프로그램 목록 정상
- 저장 정상
- AI 기능 정상
- PDF export 정상

### 조건부 배포

치명적이지 않은 일부 문제가 있어도 배포 가능할 수 있습니다.

예:
- 특정 부가 기능만 약간 느림
- 문구가 조금 어색함
- 운영 로그 확인이 더 필요함

### 배포 보류

아래는 보류입니다.

- 로그인 실패
- 저장 실패
- 업로드 실패
- AI 핵심 기능 실패
- 비교/추천 완전 비정상
- PDF export 실패

## 7. 가장 쉬운 사용 방법

배포 직전에 아래 순서대로 직접 눌러보면 됩니다.

1. 첫 화면 열기
2. 로그인
3. 프로그램 탐색
4. 프로필 저장
5. 활동 저장 + 이미지 업로드
6. AI 분석
7. PDF export

이 7개가 되면 현재 기준으로는 런칭 가능성이 높습니다.
