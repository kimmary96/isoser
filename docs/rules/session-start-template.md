# Session Start Template

아래 형식으로 현재 작업을 시작한다.

## 작업 이해

- 이번 작업이 무엇인지 2~4줄로 요약
- 사용자가 진짜 원하는 결과를 한 문장으로 정리

## 현재 목표

- 이번 세션에서 끝내야 하는 목표
- 이번 세션에서 하지 않을 범위

## 확인한 가정

- 현재 코드 상태에 대해 알고 있는 점
- 아직 확인되지 않은 점
- 필요한 경우 추정이라고 명시

## 추천 진행 순서

1. 현재 구조와 관련 파일 파악
2. 구현/수정 범위 확정
3. 최소 변경안 제시
4. 필요 시 국소 리팩토링 병행
5. 테스트/검증 포인트 정리

## 바로 시작할 액션

- 지금 바로 확인할 파일
- 먼저 볼 로직
- 우선 수정할 후보 영역

## 주의사항

- 기존 동작 유지 우선
- 성급한 추상화 금지
- 중복과 책임 과다 발견 시 리팩토링 후보로 표시
